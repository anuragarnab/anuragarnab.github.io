Research

Weakly- and Semi-Supervised Panoptic Segmentation

We present a weakly supervised model that jointly performs both semantic- and instance-segmentation -- a particularly relevant problem given the substantial cost of obtaining pixel-perfect annotation for these tasks. In contrast to many popular instance segmentation approaches based on object detectors (like Mask R-CNN), our method does not predict any overlapping instances. Moreover, we are able to segment both "thing" and "stuff" classes, and thus explain all the pixels in the image. "Thing" classes are weakly-supervised with bounding boxes, and "stuff" with image-level tags.

On the Robustness of Semantic Segmentation Models to Adversarial Attacks

Adversarial examples are arguably the greatest challenge affecting deep neural networks today, particularly if they are to be used in safety-critical systems such as autonomous vehicles. We present, to our knowledge, the first rigorous evaluation of the robustness of semantic segmentation models to adversarial attacks. We analyse the effect of different network architectures, model capacity and multiscale processing, and show that many observations made on the task of classification do not always transfer to segmentation. Furthermore, we show how mean-field inference in deep structured models and multiscale processing naturally implement recently proposed adversarial defenses.

Pixelwise Instance Segmentation with a Dynamically Instantiated Network

We propose an Instance Segmentation algorithm that produces a segmentation map where each pixel is assigned both an object class and instance identity label. Our dynamically instantiated network outputs a variable number of instances per image, and is trained end-to-end. In contrast to most competing approaches, our method requires no post-processing and reasons about all instances in the image globally (as opposed to independent proposals), and it is thus impossible for one pixel to belong to multiple instances.

Holistic, Instance-level Human Parsing

We segment the parts of objects at an instance-level, such that each pixel in the image is assigned a part label, as well as the identity of the object it belongs to. Our end-to-end trained network can thus produce state-of-the-art instance-level part segmentation, instance-level object segmentation and category-level segmentations, all in a single forward pass through the network.

Bottom-up Instance Segmentation

Traditional Scene Understanding problems such as Object Detection and Semantic Segmentation have made breakthroughs in recent years due to the adoption of deep learning. However, the former task is not able to localise objects at a pixel level, and the latter task has no notion of different instances of objects of the same class. We focus on the task of Instance Segmentation which recognises and localises objects down to a pixel level.

Higher Order Conditional Random Fields in Deep Neural Networks

Most semantic segmentation approaches now use Convolutional Neural Networks (CNN) coupled with a Conditional Random Field (CRF) to produce a structured output that is consistent with visual features of the image. We perform the CRF inference directly within the deep neural network to facilitate joint learning of the CRF and pixel-wise CNN classifier. Moreover, we incorporate two different higher order potentials, namely object detection based potentials and superpixel based potentials into our end-to-end trainable CRF model.

Audiovisual Semantic Segmentation

In this project, we use sound as an additional sensory modality to perform semantic segmentation. We obtain auditory information by tapping objects in the scene, and find that the sound emitted is characteristic of the material that the object is made of. Using this information, we are able to improve our discrimination of object classes which appear visually similar, but are made of different materials. The sound information that we collected (and release publicly), is only available in sparse locations in the scene. Our CRF model is able to effectively augment dense visual cues with sparse auditory information.

Semantic Paint

Semantic Paint is a real-time interactive system for reconstruction and object-class segmentation of 3D worlds. A user can walk into a room wearing a consumer depth camera and virtual reality headset, and reconstruct both the 3D scene and interactively segment it on-the-fly into object classes such as "chair" and "table" using an online random forest. The user can interact physically with the environment, touching or pointing at objects and using voice commands to assign labels.
